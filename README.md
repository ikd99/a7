# a7
### `python manage.py runserver`
ローカルで起動

### `python manage.py makemigrations`
変更に応じて、DBに対する処理が書かれた中間ファイル（マイグレーションファイル）をDjangoが作成

### `python manage.py migrate`
Djangoがマイグレーションファイルの内容をSQLに変換してDBに適用

### `pip install pillow`
photoを扱うためにインストール必要
